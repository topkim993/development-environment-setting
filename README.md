# development-environment-setting

1. Brew 설치
- 터미널에서 아래 명령어 실행
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
---

2. BrewFile 다운로드
- 현재 Repository에서 BrewFile을 찾아 다운로드

---
3. BrewFile 설치
- 다운로드 받은 BrewFile 경로로 이동하여 아래 명령어 실행
```
brew bundle
```
